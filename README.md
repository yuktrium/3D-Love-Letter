# 💌 3D Love Letter

A beautiful and interactive **3D Love Letter** built using **HTML, CSS, and JavaScript**. Experience a realistic envelope that gently floats, opens with a smooth animation, and reveals a heartfelt letter through elegant 3D paper unfolding effects.

## 🚀 Live Demo

https://yuktrium.github.io/3D-Love-Letter/

## ✨ Features

- 💌 Realistic 3D envelope animation
- ❤️ Smooth envelope opening effect
- 📜 Elegant paper unfolding animation
- ✨ Beautiful gradients and lighting
- 🎨 Responsive design
- ⚡ Built with pure HTML, CSS & JavaScript
- 🔧 Easy to customize

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

```bash

```
